# fileio

Stores files on the cloud for a specific time using file.io

## Installing

```shell
git clone https://github.com/unixchad/filestore.git
```

```shell
cd filestore
```

```shell
sudo make install
```

## Using the program

```shell
fileio -h
```

```shell
fileio -f foo.txt 1d
```

## Uninstalling

```shell
cd filestore
```

```shell
sudo make uninstall
```
